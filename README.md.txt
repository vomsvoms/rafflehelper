# Raffle/Bingo Number Checker

A lightweight web app to save numbers and check if a number exists—perfect for raffles, bingo, and quick lookups.

## Features
- Add single numbers or bulk add with ranges (e.g., `1-75, 101, 200-210`)
- Fast search using a Set (instant lookup)
- Persistent storage via `localStorage`
- Import/export JSON
- View count and full list

## Use
- Open `index.html` locally, or visit the GitHub Pages link once deployed.
- Bulk add via the “Bulk add” button.

## Deploy (GitHub Pages)
1. Push repo to GitHub.
2. Settings → Pages → Source: deploy from `main` (root).
3. Access at `https://<username>.github.io/<repo>/`.
